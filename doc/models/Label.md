## Label

### Fields

| Name | Type | Required | Description|
|-----------|------|----------|------------|
| id | `string \| Number` | Yes | Internal Id of the label |
| text | `string` | Yes | Text shown on label |
| color | `string` | Yes | Color identifier of the label in hex code |

### Example

```json
{
  "id": 4,
  "text": "4",
  "color": "#FFFFFF"
}
```
